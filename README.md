# StudentRecordsDemo
An archive of a web design assignment. Using HTML, CSS, JS, PHP, and MySQLite, this website has a login page with options to insert, view,  update, and delete student records.

To run this file, please open start a PHP server using the command "php -S localhost:8000", and open up login.php. Note that a separate SQL server needs to be set up before you can do insert, view, delete, and update items.
